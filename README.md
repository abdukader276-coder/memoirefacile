# MemoireFacile

MemoireFacile est une application web autonome pour les etudiants et jeunes professionnels francophones.

## Publier sur GitHub Pages

1. Decompressez le fichier ZIP.
2. Envoyez tout le contenu dans votre repository GitHub.
3. Allez dans Settings > Pages.
4. Source : GitHub Actions.
5. Dans Actions, lancez ou attendez le workflow "Deploy MemoireFacile".

Important : ne mettez pas seulement le ZIP dans GitHub. Il faut envoyer le contenu du ZIP.
